# FDA QSR Pre-Inspection Checklist

Detailed punch list per 21 CFR 820 Subpart. For each item: evidence required, common gap, remediation pattern, QMSR (effective Feb 2026) impact.

---

## Subpart B — Quality System (820.20-820.25)

### 820.20 — Management Responsibility

| Item | Evidence | Common gap |
|------|----------|------------|
| Quality policy approved | Signed policy doc | Not signed, not current |
| Quality objectives + measurable | Document with targets | Vague ("high quality") |
| Management review (annual minimum) | Meeting minutes + action items | Skipped years; no actions |
| Management representative appointed | Letter of appointment | Not appointed (or no documentation) |
| Resources adequate (people, infra) | Resource planning evidence | Resource constraints never escalated |

### 820.22 — Quality Audits

| Item | Evidence | Common gap |
|------|----------|------------|
| Internal audit schedule | Annual plan | Skipped audits |
| Audit findings documented | Audit reports | Findings; no remediation |
| Audit reports reviewed by management | Sign-off | No formal review |

### 820.25 — Personnel

| Item | Evidence | Common gap |
|------|----------|------------|
| Training records per role | LMS records | Missing for hires |
| Training effectiveness | Test results / observations | Training conducted; not assessed |
| Job descriptions match work performed | HR docs vs reality | Out-of-date |

## Subpart C — Design Controls (820.30)

Most-frequent 483 trigger area.

| Subsection | Item | Evidence | Common gap |
|------------|------|----------|------------|
| 820.30(b) | Design and development planning | Per-project plan | Not formally planned |
| 820.30(c) | Design inputs | Documented + reviewed | User needs not formally captured |
| 820.30(d) | Design outputs | Documented | Outputs don't reference inputs |
| 820.30(e) | Design review | Reviews conducted at appropriate stages | Reviews skipped |
| 820.30(f) | Design verification | V&V protocols + reports | V&V not documented or stale |
| 820.30(g) | Design validation | Validation per intended use | Validation done; not connected to inputs |
| 820.30(h) | Design transfer | Transfer procedures | Informal handoff |
| 820.30(i) | Design changes | Change control | Changes not tracked formally |
| 820.30(j) | Design History File (DHF) | Comprehensive DHF per device | Incomplete; scattered across systems |

## QMSR-specific additions (effective Feb 2026)

QMSR harmonizes 21 CFR 820 with ISO 13485:2016. Net additions:

| Area | New requirement |
|------|-----------------|
| Risk management | ISO 14971 integration mandatory |
| Software lifecycle | IEC 62304 for software-as-medical-device |
| Usability engineering | IEC 62366 |
| Post-market surveillance | More structured PMS plans |
| Clinical evaluation | Reference to clinical evaluation report |

Existing 21 CFR 820 evidence largely transfers but with new mappings.

## MDR (Medical Device Reporting, 21 CFR 803)

Separate from QSR but FDA always checks during inspection.

| Item | Evidence | Common gap |
|------|----------|------------|
| MDR procedure | Procedure | Not differentiated from general complaints |
| MDR-reportable events identified | Per complaint review | Events missed |
| 5-day report submitted (death / serious injury threat) | MedWatch submission | Late |
| 30-day report submitted (other) | MedWatch | Missed |

## Inspection-day evidence binders

Prepare ahead:

```
Binder 1: Quality System Overview
  - Quality manual
  - Quality policy + objectives
  - Org chart
  - Management review minutes (last 3 years)
  - Internal audit reports
  - Management responsibility records

Binder 2: Per-Device DHF
  - One binder per device in scope

Binder 3: CAPA log
  - All CAPAs (open + closed) for inspection period
  - Per CAPA: RCA, action, evidence, effectiveness verification

Binder 4: Complaints + MDR
  - Complaint log
  - Per complaint: investigation, disposition, MDR (if applicable)

Binder 5: Production + Process
  - Process validations
  - Equipment calibration / PM
  - Environmental monitoring
  - Acceptance records

Binder 6: Suppliers
  - Approved supplier list
  - Per critical supplier: qualification, agreement, evaluations
```

## Cheat sheet

| Question | Answer |
|----------|--------|
| First call when inspection announced? | QA Lead + outside FDA counsel (for advice) |
| First call when 483 received? | Outside FDA counsel within 24 hours |
| First call when WL received? | Outside FDA counsel + CEO + GC immediately |
| 483 response deadline? | 15 business days |
| WL response deadline? | 15 business days |
| Inspection length? | Typically 3-10 days; routine vs for-cause |
| Most-cited subpart? | Subpart J (CAPA) — design controls (Subpart C) next |
| QMSR transition timing? | Effective Feb 2026; harmonize prep now |
