## Step 3: Calibration gaps

Wherever you and your manager disagree by >=1 step, that's a **calibration gap**. Examples:

- You: "At bar for Strategy at target level". Manager: "Below bar". -> Calibration gap.
- You: "Above bar for Communication". Manager: "At bar". -> Mild calibration gap.

| Dimension | Your score | Manager score | Gap size |
|-----------|-----------|----------------|----------|
| Product Sense | | | |
| Execution | | | |
| Leadership | | | |
| Strategy | | | |
| Communication | | | |

**For each calibration gap, ask:**

1. What specific behavior is missing in my manager's view?
2. What evidence would close the gap (artifact, project, observable change)?
3. What is the time horizon to close it?
