## Agenda

### 0:00 - 0:30 | Reflect: Score the OKRs (30 min)

For each KR, score 0.0 to 1.0 using the table below:

| Score | Interpretation |
|-------|----------------|
| 0.0 - 0.3 | Failed to move; hypothesis or execution was wrong |
| 0.4 - 0.6 | Partial progress; mixed signal |
| 0.7 - 0.9 | Strong outcome (Wodtke's target zone) |
| 1.0 | Maximum success (warning: was the target sandbagged?) |

**Average team score:** [X.X]

**Wodtke heuristic check:**
- If average > 0.9: were the targets sandbagged?
- If average < 0.5: were the targets unrealistic, or was execution off?
- Target zone over multiple quarters: 0.6-0.7

### 0:30 - 1:00 | Reflect: What Did We Learn? (30 min)

Round-robin -- each participant shares:

#### About the customer

- One thing we learned about what the customer wants, needs, or struggles with
- Source: which interview, which data, which signal

#### About the product

- One thing we learned about what works (or does not) in the product
- Source: which experiment, which release, which data

#### About the team / process

- One thing we learned about how the team operates
- One thing we will start, stop, or continue

## Optional: Anonymous "What did you not say?" round (10 min addition)

If team dynamics suggest unspoken concerns, run a short anonymous round:

- Each participant writes one thing they did not feel comfortable saying out loud
- Facilitator reads them aloud anonymously
- Team discusses themes

Use sparingly -- not every retro needs this, but it can surface issues that block honest scoring.
