### 1:30 - 2:15 | Decide: Commit to OKRs (45 min)

**Vote on 1-3 Objectives:**

- Each participant gets 1 vote (3 if multiple Os allowed)
- Discuss top-voted Objectives; final commit per Objective requires team consensus

**For each committed Objective, define 3-5 Key Results:**

```markdown
## Objective: [...]

> Why this Objective serves the vision: [...]

### KR1: [Metric] from [baseline] to [target] by [date]
- Owner: [...]
- Confidence at commit: [1-10]
- Validation source: [...]

### KR2: [...]

### KR3: [...]
```

**Apply SMART check (from `execution/brainstorm-okrs/`):**

- [ ] Specific
- [ ] Measurable (baseline + target + deadline)
- [ ] Achievable (~70% confidence at commit per Wodtke)
- [ ] Relevant (traces to Objective and vision)
- [ ] Time-bound

---
