# Customer Profile Worksheet

A standalone worksheet to capture jobs, pains, and gains for a single segment. Use this *before* attempting the Value Map. The worksheet exists to slow you down: most teams jump to solutions before understanding the customer's world.

## Validation Checklist

Before moving to the Value Map worksheet:

- [ ] At least 5 customer interviews per segment completed
- [ ] Each job statement uses customer language (verbatim quotes attached)
- [ ] Each pain statement uses customer language (verbatim quotes attached)
- [ ] Each gain statement uses customer language (verbatim quotes attached)
- [ ] Jobs ranked by importance x frequency
- [ ] Pains ranked by severity x frequency
- [ ] Gains ranked by desirability
- [ ] At least 1 Unexpected gain (delighter) identified
- [ ] Top 3 jobs, pains, and gains agreed on by the product trio (PM + Design + Eng)

If any box is unchecked, return to interviews before drafting the Value Map.
